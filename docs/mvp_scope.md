# Périmètre MVP

## Inclus

- Auth (signup/login/logout)
- Profil utilisateur (handle, bio, avatar)
- Posts texte + image
- Home feed (following uniquement)
- Profile feed
- Follow / unfollow
- Like / unlike
- Modération basique (flag + soft-hide)
- Interface admin

## Exclus (hors MVP)

- Politique
- Religion
- Discours haineux
- Reposts / retweets
- DMs
- Trending / recommandations
- Vidéo
- Recherche avancée
